⚽ WM 2026 Tippspiel – Einrichtung

⚙️ Einmalige Einrichtung – danach einfach den Link an alle Freunde schicken!
1

Supabase Projekt anlegen

Auf supabase.com kostenlos registrieren → Neues Projekt → Name: wm2026, Region: EU (Frankfurt) oder EU (Stockholm).

2

SQL ausführen

Im SQL Editor die Datei setup.sql einfügen und ausführen. Tabellen, RLS-Policies, pg_cron-Job und Ranglisten-View werden erstellt.

3

Zafronix API-Key holen

Auf api.zafronix.com/signup kostenlosen Key holen (250 Req/Tag, reicht locker). Key unter Supabase → Edge Functions → Secrets als ZAFRONIX_API_KEY speichern.

4

Edge Function deployen

Die Datei supabase/functions/fetch-scores/index.ts mit der Supabase CLI deployen:

supabase functions deploy fetch-scores --project-ref DEIN_REF

Danach in setup.sql die Zeile mit net.http_post anpassen (Projekt-Ref + Service-Role-Key eintragen) und ausführen.
